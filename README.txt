-- About perceptron -- 
It is a model which a single neuron. Here, it is used for a simple classification problem. The problem deals with provided Height, Width and Thickness of a bed, to classify it as either Small Bed or Large bed. One can find dataset in the repo, named "bed.csv".

Download the repo and you can train the model by providing simple commands.

-- Commands -- 
python perceptron.py train
python perceptron.py test
